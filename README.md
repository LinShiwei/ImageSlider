## Installation
### Carthage

Using [Carthage](https://github.com/Carthage/Carthage):

```
github "LinShiwei/ImageSlider"
```

### Manually
* Just include the two ｀.swift｀ files found on the `ImageSlider` folder on the demo Xcode project.
